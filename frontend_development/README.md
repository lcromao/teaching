# Material de Apoio para curso de FrontEnd Development

Este repositório agrega materiais de apoio, exemplos práticos e guias para o curso de FrontEnd Development, focando em boas práticas, frameworks populares e técnicas modernas de desenvolvimento web.

## Criando o ambiente de desenvolvimento no Github Codespaces
1. Crie sua conta no [GitHub](https://github.com/).
2. Crie um novo repositório dedicado ao curso (não precisa ser público).
3. No repositório, clique no botão **Code** e selecione a aba **Codespaces**.
4. Clique em **Create codespace on main**.
5. Aguarde a configuração do ambiente (pode levar alguns minutos).

## Extensões úteis para produtividade no VSCode

- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer): Para um servidor local com recarregamento automático.
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode): Para formatação de código.
