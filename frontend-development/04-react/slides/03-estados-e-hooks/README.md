# Apresentação - Estados e Hooks no React

Apresentação interativa sobre a aula 3 de React: eventos, `useState`, hooks customizados, inputs controlados, `useRef`, `useEffect` e modelos mentais de decisão.

## 🚀 Como executar

1. Instalar as dependências:
```bash
npm install
```

2. Iniciar o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abrir o navegador no endereço indicado (geralmente `http://localhost:5173`)

## 📋 Funcionalidades

- **34 slides** sobre estados e hooks no React
- Navegação por teclado: `←`, `→`, `Space`, `PageUp`, `PageDown`
- Navegação por botões na interface
- Barra de progresso
- Design responsivo

## 📚 Conteúdo da apresentação

1. Eventos no React
2. State e `useState`
3. Visão geral de hooks
4. Regras dos hooks
5. Hooks customizados
6. Inputs controlados
7. Estado em ação: compartilhamento, condicionais e listas
8. `useRef` e regra prática `state` vs `ref`
9. `useEffect` (parte 1) e seus modelos mentais
10. Regra de bolso `state/ref/effect`
11. Resumo e próximos passos

## 🛠️ Tecnologias

- React 18

- CSS Modules
